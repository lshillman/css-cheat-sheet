CSS Cheat Sheet

Some useful CSS snippets to copy for your own use

Rounded image

Style image in the round

img {
    border-radius: 50%
}

Box Transition Glow

Use transition and box shadows to glow on hover

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}

Linear Gradients

This will create a background that is a gradient from top to bottom

.linear-gradient-background {
    background-image: linear-gradient(
        rgba(232, 102, 236, 0.3) 0%
        rgba(232, 102, 236, 0.6) 100%
    );
}

Overlay Card with Title

Use position properties and negative margins to raise elements higherthat their natural starting point.

.card-header {
    position: relative;
    margin-top: -20px;
}

Flexbox Column

This creates a Flexbox column layout

.column {
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a 10-column layout using CSS Grid

.grid {
    display: grid;
    width: 100%
    grid-template-columns: repeat(10, 1fr);
}
